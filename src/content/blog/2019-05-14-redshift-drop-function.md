## 概要

- DROP FUNCTION で関数名と引数の指定が必要。

### 対応

- 作成sql

```sql
CREATE FUNCTION f_hoge(tag_str VARCHAR, tag_number int)
RETURNS bool
IMMUTABLE AS $$
try:
  target_arr = tag_str.split(",")
  return '_' in target_arr[tag_number]
except: return False
$$ LANGUAGE plpythonu;
```


- 削除sql

```sql
DROP FUNCTION f_hoge(tag_str VARCHAR, tag_number int)
```
